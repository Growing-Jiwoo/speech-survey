# components/survey/ — 참여자(아동) 검사 화면 컴포넌트

만 6-7세 아동이 교사 보조 하에 사용하는 화면. **큰 터치 타깃, 즉각적 피드백, 오터치 방지,
평가 신호(칭찬/실망) 비노출**이 설계 원칙이다. `app/survey/page.tsx`가 문항 타입에 따라
아래 컴포넌트를 골라 렌더한다.

| 파일 | 문항/역할 |
|---|---|
| `MicCheck.tsx` | 검사 시작 전 마이크 확인 — "안녕하세요" 발화의 peak로 판정, 실패 종류별(거부/미지원/실패) 안내. 성공 후 재확인 가능 |
| `RecordingItem.tsx` | 녹음 문항(낱말 30초/문장 40초) — 녹음 종료 즉시 업로드, 저장 중/완료/저음/자동종료 피드백, 실패 시 부모(RetryBanner)로 위임 |
| `WritingItem.tsx` | 낱말 쓰기 문항 — 검사자가 예/아니오 선택(선택 전 [다음] 비활성) |
| `ChecklistItem.tsx` | 검사자 체크리스트 — 발달 영역 다중 선택('특이사항 없음'과 상호 배타) |
| `RetryBanner.tsx` | 업로드 실패 녹음 재시도 배너 — 문항을 이동해도 유지, 연타 중복 업로드 가드 |

녹음 자체(MediaRecorder·레벨미터·타이머)는 `hooks/useRecorder.ts`가 담당한다.
