# docs/ — 설계 문서 아카이브

`superpowers/` 아래에 기능 단위의 설계·구현 계획 문서가 날짜순으로 쌓여 있다
(superpowers 워크플로: brainstorming → specs 설계 → plans 구현 계획 → 구현).

- `superpowers/specs/` — 기능 설계 문서(요구사항·의사결정·트레이드오프)
- `superpowers/plans/` — 구현 계획(작업 단위 분해)

**이 문서들은 작성 시점의 스냅샷**이다 — 현재 동작과 다를 수 있으므로 코드가 항상 우선한다.
새 기능의 맥락(왜 이렇게 만들었는지)을 찾을 때 참고.
